Początki - Microsoft FrontPage i Blogspot

Swoje pierwsze kroki stawiałem w szkole podstawowej tworząc proste szablony w programie Microsoft FrontendPage. Następnym krokiem była krótka przygoda z platformą blogspot (blogger), gdzie po raz pierwszy zafascynowały mnie przykuwające uwagę skórki stron typu Magazine. Szybko przeszedłem na system zarządzania tręścią o nazwie Wordpress, który rozwiązywał problem zależności od serwisu Google, który w każdej chwili mógł zostać usunięty bez słowa ostrzeżenia.

Wordpress i Page Buildery

Z Wordpressem jestem hobbistycznie związany od około 2009 roku. Przez większość tego czasu, głównie obserwowałem, jak rozwija się ta platforma oraz Page Builder, które zainteresowały mnie możliwością wizualnej edycji stron bez znajomości języków programowania. Niestety zawsze okazywało się, że w każdym Page Builderze czegoś mi brakowało. Często były to bardzo podstawowe funkcje więc z ciekawością obserwowałem nowe projekty, zastanawiając się czy kolejny Page Builder będzie oferował większe możliwości od poprzedniego. Najpierw poznałem SiteOrigin Page Builder, a następnie Visual Composer, Divi, Elementor, Beaver Builder i kilka innych, których nazw nie pamiętam.

Kreator stron Wordpress - Oxygen Builder

Po wielu doświadczeniach z Page Builderami, trafiłem na kreator stron o nazwie Oxygen Builder. To plugin przypominający inne Page Buildery, jednak jego zamysł jest nieco inny. Jest to narzędzie dla deweloperów, które daje znacznie większe możliwości, jednakże wymaga dobrej znajomości takich języków jak: CSS, Javascript, PHP i HTML. Poznając Oxygena, zrozumiałem, że kluczem do całkowitej swobody jest znajomość języków niezbędnych do tworzenia stron. Żadne narzędzie nie daje takich możliwości jak znajomość kodowania.

I tak postanowiłem rozpocząć swą przygodę z nauką języków programowania 🐱‍💻
